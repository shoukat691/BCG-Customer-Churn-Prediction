**BCG X Data Science Virtual Experience - Customer Churn Prediction**

 Project Overview
This project was completed as part of the BCG X Data Science Job Simulation. The goal was to build a predictive model to help a utility company identify customers who are likely to churn (leave).

**Objective**
To develop a high-accuracy machine learning model that predicts customer churn based on historical consumption, price margins, and customer data.

**Methodology**
Exploratory Data Analysis (EDA):** Analyzed customer behavior and consumption patterns.
Data Preprocessing:** Handled missing values, encoded categorical features, and performed feature scaling.
- **Model Selection:** Used a **Random Forest Classifier** to handle complex data relationships.
- **Evaluation:** Split the data into 80% training and 20% testing sets.

**Key Results**
- **Accuracy:** The model achieved an **Accuracy Score of 90%**.
- **Top Drivers:** The most important factors for churn were identified
